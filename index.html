Software Engineering
IBM DB2

IBM DB2

Description

For my co-op work term, I had the opportunity to work at IBM Canada Ltd. as a DB2 Continuous Engineering Developer. During my time at IBM, I took part in the design, testing, and implementation of Shadow tables. Shadow tables are column-organized copies of row-organized tables, implemented as materialized query tables (MQTs) maintained by replication. This allows users to get the performance benefits of BLU acceleration for analytic queries in an OLTP environment.

Challenges

Learnt to navigate through the millions of lines of code and thousands of code files for debugging and implementation. Collaborated on projects alongside fellow developers located in different countries around the world.

Technologies

C/C++, SQL, Perl, Shell-scripting, DB2 LUW, Hummingbird Exceed, Emacs, Clearcase, TotalView, VNC Viewer, Windows, Unix, Linux.

3D Animation
TRON Logo

TRON Logo

Description

Designed and animated a moving logo inspired by Disney's TRON films. Using Autodesk 3ds Max, I created materials and modelled the logo to resemble the source material. Followed by lighting and rendering, I added some key framing and path animation to control the position and rotation of the object. Finally, I used Adobe After Effects to add some final visual effects to complete the composition.

Challenges

It was challenging to figure out how I wanted to animate the object initially. I wanted to create something simple and smooth, yet followed the theme of the source material. I was quite happy with how it turned out in the end.

Technologies

Autodesk 3ds Max, Adobe After Effects, Adobe Photoshop.
